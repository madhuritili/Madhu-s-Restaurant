# Madhu's Restaurant – Web Page Project

## Overview
A responsive **restaurant web page** built using **HTML5** and **CSS3**, showcasing the restaurant’s menu, ordering system, feedback form, location map, and contact information. Designed for a modern, user-friendly browsing experience across **desktop devices**.

---

## Features
- **Home Section** – Welcome message and tagline.  
- **About Us** – Information about the restaurant.  
- **Menu Section** – Displays various dishes with images.  
- **Order Section** – Form to select dishes, quantity, delivery address, and confirm order.  
- **Restaurant Video** – Embedded video showcasing the restaurant.  
- **Location Section** – Embedded Google Map for restaurant location.  
- **Feedback Section** – Users can submit ratings.  
- **Contact Section** – Email and phone contact information.  
- **Responsive Design** – Optimized for desktop browsing.

---

## Technologies Used
- HTML5  
- CSS3  
- Embedded Google Maps and video

1. Clone or download the repository.
